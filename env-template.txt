# ============================================
# Lynx 环境变量配置模板
# ============================================
# 使用方法：
# 1. 复制此文件内容到 .env.local
# 2. 替换 192.168.0.100 为你的本机 IP
# 3. 重启开发服务器
# ============================================

# 如何获取本机 IP：
# macOS/Linux: ifconfig | grep "inet " | grep -v 127.0.0.1
# Windows: ipconfig
# 或运行: node mock/server.cjs 查看局域网地址

# ============================================
# 后端 API 地址（完整 URL）
# ============================================
# Lynx 真机环境必须使用完整 URL（包含 http:// 和端口）
# 不能使用 localhost，必须使用真实 IP
VITE_API_HOST=http://192.168.0.100:4000

# ============================================
# 前端服务器地址（完整 URL）
# ============================================
# 用于访问静态资源（如图片）
# 端口应该与 npm run dev 启动的端口一致（通常是 3000）
VITE_FRONTEND_URL=http://192.168.0.100:3000

# ============================================
# 注意事项：
# ============================================
# 1. 修改后需要重启开发服务器（Ctrl+C 然后 npm run dev）
# 2. .env.local 不会被提交到 Git（个人配置）
# 3. 确保前后端端口与实际运行的端口一致
